### ONLY FOR ANDROID BY NOW

## USAGE

### Add the package to your npm dependencies

```
npx expo install expo-cdnbye
```

### Config the plugins token

```json
#app.json

{
  "expo": {
  ...
    "plugins": [
      [
        "expo-cdnbye",
        {
          "token": "CDNBYE_TOKEN"
        }
      ]
    ]
  ...
  }
}
```
